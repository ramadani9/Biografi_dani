Isi masukan Untuk Saya Pribadi ?

 

Nama saya Nur Ramdani subagja, biasa di panggil dani saya lahir 30 desember 1998. saya pernah memiliki latar belakang pemograman
sewaktu saya masih bersekolah tetapi 2 tahun terakhir ini saya sama sekali tidak update tentang dunia IT, kenapa saya ingin belajar pemograman lagi ?
karena saya ingin meningkatkan lagi pengetahuan saya tentang pemograman itu sendiri dan saya juga ingin berkarir di dunia pemograman. tujuan saya
setelah menyelesaikan pemograman ini ? saya ingin mencari kerja sesuai dengan ilmu yang saya miliki saat mengikuti pembelajaran front end basic.
dan hobby saya : sepak bola, berenang, dan olahraga lain-nya.
